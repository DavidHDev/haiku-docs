# useToggle()

The `useToggle()` hook can toggle between a set of two possible values and automatically update the state with the new value.

### Import

```jsx
import { useToggle } from 'react-haiku';
```

### Usage

import { UseToggleDemo } from '../../demo/UseToggleDemo.jsx';

<UseToggleDemo />

```jsx
import { useToggle } from 'react-haiku';

export const Component = () => {
    const [theme, toggleTheme] = useToggle('dark', ['dark', 'light']);

    return (
        <>
            <b>{`Theme: ${theme}`}</b>
            <button onClick={() => toggleTheme()}>Toggle!</button>
        </>
    );
}
```

### API

This hooks accepts the following arguments:
- `initialValue` - the initial preferred value for the toggle state
- `options` - must be an array with strictly 2 elements and must also contain the value set for `initialValue`
